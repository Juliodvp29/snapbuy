# SnapBuy

> Find what you see. Buy the best.

Mobile application that allows you to photograph any object and receive
Purchase recommendations ordered by price, quality and ratings.

##Stack

- **App:** Kotlin + Jetpack Compose
- **Backend:** Python + FastAPI
- **AI:** CLIP (OpenAI)

## Structure

- `/app` — Android application
- `/backend` — REST API
- `/docs` — Technical documentation
